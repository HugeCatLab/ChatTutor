# @chat-tutor/web

This is the web application for ChatTutor, based on Vue 3 and TypeScript.

## Usage

```bash
pnpm dev
```

## Build & Run

```bash
pnpm build
pnpm start
```

## Link API Server

You need to link the API server in environment variables.

```bash
VITE_API_BASE_URL=http://localhost:8002
```
