# @chat-tutor/ggb-applet

This is a type declaration for the GeoGebra Apps API.

About Applet: https://geogebra.github.io/docs/reference/en/GeoGebra_Apps_API/

## Usage

```html
<script src="https://www.geogebra.org/apps/deployggb.js"></script>
```

```typescript
// global.d.ts
/// <reference types="@chat-tutor/ggb-applet" />
```
